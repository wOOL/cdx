Release Notes 9.6.3

English

  • Improvements in digital drill guide wizard:
  • Added option to display sleeves during definition of bone support regions (#4014).
  • Wizard now persistently stores selected segmentation for bone support regions (#4015).
  • Diameter and height of inspection windows can now be modified using mouse wheel - analogously to sleeve body size (#4065).
  • Sleeve calibration matrix can now be generated via the context menu of a planned sleeve (#3599).
  • Zoom with mouse wheel enabled for all 3D views in the wizard (#4492).
  • In virtual planning export, segmentation export quality has been improved in many cases (#4499).
  • Solved an issue when transferring implants via caseXchange (#4503).
  • Several minor bugs fixed.

Deutsch

  • Verbesserungen am Assistenten für die digitale Bohrschablone:
  • Eine Option zum Anzeigen von Hülsen beim Definieren von Knochenauflageflächen wurde hinzugefügt (#4014).
  • Im Designer für Knochenauflageflächen wird nun die ausgewählte Segmentierung dauerhaft gespeichert (#4015).
  • Sichtfenster können nun mittels des Mausrades in Durchmesser und Höhe verändert werden - analog des Verhaltens bei Hülsenfassungen (#4065).
  • Die Hülsenkalibrierungsmatrix kann nun über das Kontextmenü einer geplanten Hülse generiert werden (#3599).
  • Zoomen über das Mausrad wurde für alle 3D-Ansichten des Assistenten aktiviert (#4492).
  • Im virtuellen Planungsexport wurde die Qualität exportierter Segmentierungen für viele Fälle verbessert (#4499).
  • Ein Problem beim Übertragen von Implantaten über caseXchange wurde behoben (#4503).
  • Weitere kleinere Fehler beseitigt.
