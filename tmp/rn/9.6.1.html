Release Notes 9.6.1

English

  • (#4400) New stereo 3D mode “side-by-side” for use with 3D displays.
  • (#4436) In digital drill guide wizard, default settings were changed to not use large connectors and add labels embossed.
  • (#2129) Added option to set advanced DICOM import as default.
  • (#4402) When aligning views to implants a bigger view area is displayed.
  • (#4309) Problem with adding measurements to a planning containing abutments solved.
  • (#4443) Distance of implant move in plan comparison dialog now correctly calculated.
  • (#4385) Sleeve adjustment dialog multi selection consistency fixed.
  • (#4282) In Virtual Planning Export, the export settings for segmentations can be adjusted.
  • New sleeve systems added: C.Hafner, Sweden&Martina and BioHorizons.
  • Several minor bugs were fixed.

Deutsch

  • (#4400) Neuer 3D-Stereo-Modus “Side-by-side” für kompatible 3D-Bildschirme.
  • (#4436) Standard-Einstellungen für “Große Konnektoren” und “Text erhaben” im Assistent für digitale Bohrschablonen geändert.
  • (#2129) Option hinzugefügt, um den DICOM-Import immer im erweiterten Modus zu starten.
  • (#4402) Die Funktion “Ansicht auf Implantat zentrieren” zeigt nun einen größeren Ausschnitt.
  • (#4309) Problem behoben, welches verursachte, dass das Hinzufügen von Messungen zu Plänen, die ein Sekundärteil enthielten, fehlschlug.
  • (#4443) Angezeigte Distanz der Implantate im Planvergleich korrigiert.
  • (#4385) Probleme mit der Mehrfachauswahl im Hülsendialog behoben.
  • (#4282) Im virtuellen Planungsexport können nun die Einstellung für den Export von Segmentierungen angepasst werden.
  • Neue Hülsensysteme hinzugefügt: C.Hafner, Sweden&Martina und BioHorizons.
  • Weitere kleinere Fehler beseitigt.
