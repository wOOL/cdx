Release Notes 9.7.5

English

  • coDiagnostiX does now offer information on whether a dataset has been unlocked for production, when this was done and by which license number. This information can be found in the Plan menu under “Check lock state for production”.
  • In Synergy, the doctor’s name which is entered in DWOS/CARES is now displayed in the inbox to make it easier to identify the case if several cases have been shared.
  • For digital drill guides, the diameter and height of inspection windows can now be set down to 0.4 millimeters. This allows advanced guide features like flushing channels (small diameter) or slots for drill handles (low height).
  • When a dataset is locked after archiving, it can now be unlocked even if the corresponding message had been dismissed. In the main menu, go to Patient > Remove archive lock.
  • Added support for the new Straumann 2.2 mm sleeve.
  • Dental Wings sleeve system renamed as Meisinger.
  • Revised the surgical protocol for Straumann 2.8 mm sleeves and 8 mm BLT implants.
  • Revised 3DDX and Neoss sleeve systems.

Deutsch

  • coDiagnostiX bietet nun Informationen darüber, ob ein Datensatz für die Produktion freigeschaltet wurde, wann dies geschehen ist und durch wen. Diese Informationen sind im Menü Plan unter “Freigabestatus für Produktion überprüfen” zu finden.
  • In Synergy wird nun der in DWOS/CARES angegebene Behandlername im Synergy-Posteingang angezeigt. Dies vereinfacht das Identifizieren von verschiedenen Fällen.
  • Bei digitalen Bohrschablonen können Höhe und Durchmesser von Sichtfenstern bis 0,4 mm herunter gesetzt werden. Dies erlaubt Funktionen wie Spülkanäle (geringer Durchmesser) und Fassungen für Bohrlöffel (geringe Höhe).
  • Wenn ein Datensatz nach dem Archivieren gesperrt ist, kann die Sperre nun aufgehoben werden, auch wenn die entsprechende Nachricht bereits ausgeblendet wurde. Klicken Sie im Hauptmenü auf Patient > Archivsperre aufheben.
  • Unterstützung für die neue Straumann 2,2mm-Hülse wurde hinzugefügt.
  • Das Dental-Wings-Hülsensystem wurde in Meisinger umbenannt.
  • Das chirurgische Protokoll für Straumann 2,8 mm Hülsen und 8 mm BLT Implantate wurde überarbeitet.
  • Die Hülsensysteme von 3DDX und Neoss wurden überarbeitet.

Italiano

  • coDiagnostiX offre ora informazioni se un set di dati è stato sbloccato per la produzione, quando è stato fatto e da quale numero di licenza. Queste informazioni si possono trovare nel menù Pianificazione sotto Verifica situazione di blocco per la produzione.
  • In Synergy, il nome del medico che è iscritto nel DWOS / CARES viene ora visualizzato nella casella inbox per rendere più semplice l'identificazione del caso quando diversi casi sono stati condivisi.
  • Per le guide digitali di foratura, il diametro e l'altezza delle finestre di ispezione possono essere impostate fino a 0,4 millimetri. Questo permette funzioni di guida avanzate come canali di raffreddamenti (piccolo diametro) o slot per manipoli di foratura (altezza ridotta).
  • Quando un insieme di dati viene bloccato dopo l'archiviazione, ora può essere sbloccato anche se il messaggio corrispondente era stato eliminato. Nel menù principale, andare a Paziente > Rimuovere blocco archivio.
  • E’ stata aggiunta la nuova boccola Straumann 2,2 millimetri.
  • Il Sistema di boccole Dental Wings è stato rinominato come Meisinger.
  • Rivisti i sistemi di boccole 3DDX e Neoss.
