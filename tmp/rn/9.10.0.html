Release Notes

coDiagnostiX 9.10

English

coDiagnostiX EASY

The coDiagnostiX EASY step-by-step implant planning wizard has been completely reworked. It is easy to learn and use yet covers a wide range of treatment scenarios. It offers the full workflow from dataset alignment to export of surgical guides.

Surgical Guide Designer

3D objects can now be added to or subtracted from the guide. This enables adding e.g. a digital prosthetic setup or a scanned denture to the guide, in order to register it to the bite during surgery.

Interface to Mininavident DENACAM Implant Surgery

With coDiagnostiX 9.10, your planning data can be exported to use within DENACAM Implant Surgery, a 3D navigation system for implantology.

New sleeve systems

  • Biomet 3i Tapered Navigator
  • TBR GUIDED CLASSIC/MCI
  • Titaniumfix
  • Extension for Neodent GM

Surgical protocols

  • Added support for Straumann BLT 2.9.
  • Improved readability by changing the way of displaying color codes.

Further improvements

  • coDiagnostiX now supports the Polygon File Format (PLY) as an importable file format. It offers support for e.g. colored intra-oral scans.
  • The DICOM import now supports the ISO 2022 IR 13 string format to improve compatibility with Japanese characters.
  • In views and on printouts, the readability of measurements was improved by making the font size adjustable.
  • A button to quickly add measurements can be added to the toolbar.
  • A button to quickly switch view configurations can be added to the toolbar.
  • The tooth position of implants can now be displayed in the views.
  • New option to disable caseXchange and Order Management notifications.
  • Performance optimizations

Known Issues

  • coPeriodontiX is temporarily not accessible in version 9.10.

German

coDiagnostiX 9.10

coDiagnostiX EASY

coDiagnostiX EASY, Schritt-Für-Schritt-Assistent für Implantat-Planungen, wurde komplett überarbeitet. Er ist einfach in der Anwendung und dennoch unterstützt er eine Vielzahl von Anwendungsszenarios. Er bietet einen vollständigen Arbeitsablauf von der Vorbereitung des Datensatzes bis zum Export chirurgischer Schablonen.

Designer für chirurgische Schablonen

3D-Modelle können jetzt der Schablone hinzugefügt oder von ihr abgezogen werden. Dies erlaubt zum Beispiel das Hinzufügen einer prothetischen Aufstellung oder einer gescannten Prothese zur Schablone, um sie während der Operation am Gegenbiss auszurichten.

Schnittstelle zu Mininavident DENACAM Implant Surgery

Mit coDiagnostiX 9.10 können Sie Ihre Planungsdaten zu DENACAM Implant Surgery übertragen, einem 3D Navigationssystem für die Implantologie.

Neue Hülsensysteme

  • Biomet 3i Tapered Navigator
  • TBR GUIDED CLASSIC/MCI
  • Titaniumfix
  • Erweiterung für Neodent GM

Chirurgische Protokolle

  • Unterstützung für Straumann BLT 2.9 hinzugefügt.
  • Die Lesbarkeit wurde verbessert, in dem die Darstellung von Farbcodes angepasst wurde.

Weitere Verbesserungen

  • coDiagnostiX unterstützt jetzt das Polygon File Format (PLY) als importierbares Dateiformat. Dies wird Beispielsweise für farbige Intraoral-Scans verwendet.
  • Der DICOM-Import unterstützt jetzt ISO 2022 IR 13 Zeichenketten um die Kompatibilität zu Japanischen Schriftzeichen zu verbessern.
  • Die Lesbarkeit von Messwerten wurde in den Ansichten und auf den Druckprotokollen verbessert, indem die Schriftgröße veränderbar gemacht wurde.
  • Eine Schaltfläche, um schnell Messungen hinzuzufügen, kann in die Werkzeugleiste eingefügt werden.
  • Eine Schaltfläche, um schnell zwischen Ansichts-Einstellungen zu wechseln, kann in die Werkzeugleiste eingefügt werden.
  • Die Zahnposition eines Implantates wird jetzt auf Wunsch in den Ansichten angezeigt.
  • Neue Option, mit deren Hilfe sich caseXchange- und Auftragsverwaltungs-Mitteilungen deaktivieren lassen.
  • Geschwindigkeitsoptimierungen

Bekannte Probleme

  • coPeriodontiX ist in Version 9.10 vorübergehend nicht verfügbar.

Italian

coDiagnostiX 9.10

coDiagnostiX EASY

La procedura guidata per la pianificazione implantare con coDiagnostiX EASY è stata completamente rielaborata. È di facile apprendimento ed utilizzo ma copre una vasta gamma di scenari di trattamento. Offre il flusso di lavoro completo, dall'allineamento del set di dati all'esportazione di guide chirurgiche.

Progettazione di guide chirurgiche

Ora gli oggetti 3D possono essere aggiunti o sottratti alla guida. Ciò consente di aggiungere per es. un setup protesico digitale o una protesi scansionata alla guida per registrarlo al bite durante l'intervento.

Interfaccia con Mininavident DENACAM Implant Surgery

Con coDiagnostiX 9.10, i dati di pianificazione possono essere esportati per l'uso all'interno di DENACAM Implant Surgery, un sistema di navigazione 3D per implantologia.

Nuovi sistemi di boccole

  • Biomet 3i Tapered Navigator
  • TBR GUIDED CLASSIC / MCI
  • Titaniumfix
  • Estensione per Neodent GM

Protocolli chirurgici

Supporto per Straumann BLT 2.9 aggiunto.

Migliore leggibilità grazie alle modifiche nel modo in cui vengono visualizzati i codici colore.

Ulteriori miglioramenti

  • coDiagnostiX ora supporta Polygon File Format (PLY) come formato di file di importazione.
  • Questo formato offre supporto, ad esempio, per le scansioni intraorali colorate.
  • L'importazione DICOM ora supporta il formato di stringa IR 13 ISO 2022 per migliorare la compatibilità con i caratteri giapponesi.
  • La leggibilità delle misurazioni nelle viste e nelle stampe è stata migliorata rendendo la dimensione del carattere regolabile.
  • Un pulsante per aggiungere rapidamente misurazioni può essere aggiunto alla barra degli strumenti.
  • Un pulsante per cambiare rapidamente le configurazioni di visualizzazione può essere aggiunto alla barra degli strumenti.
  • La posizione degli impianti secondo la numerazione dentale può ora essere visualizzata nelle viste.
  • Nuova opzione per disabilitare le notifiche di CaseXchange e Gestione ordini.
  • Ottimizzazioni delle prestazioni.

Problemi noti

coPeriodontiX è temporaneamente non accessibile nella versione 9.10.
