Release Notes

coDiagnostiX 9.9

English

Improved communication

  • Synergy now updates the clinical model in coDiagnostiX if it is changed in DWOS/CARES Visual.
  • In the surgical guide designer, the caseXchange order number is now available as a labeling preset. The patient case has to be opened from the Order Management to access this preset.
  • caseXchange notification emails now contain more information about the sender of the patient case.

Sleeve systems

  • Support for Hahn™ Tapered Implant Guided Surgery system added.
  • Support for Medentika Guided Surgery system added.
  • Support for Straumann PURE implants added to the Straumann Guided Surgery system.

Further improvements

  • When editing abutments, the dialog box asking to which implant the abutment should be added does now show up when relevant only.
  • DICOM Import now offers an image clipping preset to process images without scaling.
  • Support for patient datasets planned with coDiagnostiX version 9.0 or older was improved.
  • Instabilities have been fixed in caseXchange, virtual planning export, abutment planning and PDF export.

German

Verbesserte Kommunikation

  • Synergy aktualisiert nun das klinische Modell in coDiagnostiX, wenn es in DWOS/CARES Visual modifiziert wird.
  • Im Designer für chirurgische Schablonen steht nun die caseXchange-Bestellnummer als Beschriftungsvorlage zur Verfügung. Die Planung muss dafür aus dem Order Management geöffnet werden.
  • Informations-E-Mails zu caseXchange enthalten nun mehr Information über den Absender des Patientenfalls.

Hülsensysteme

  • Unterstützung für das System Hahn™ Tapered Implant Guided Surgery hinzugefügt
  • Unterstützung für das Medentika Guided Surgery System hinzugefügt
  • Unterstützung für Straumann PURE Implantate zum Straumann Guided Surgery System hinzugefügt.

Weitere Verbesserungen

  • Beim Bearbeiten von Abutments wird die Abfrage, zu welchen Implantaten die Abutments hinzugefügt werden sollen, jetzt nur noch in relevanten Fällen angezeigt.
  • Der DICOM-Import bietet jetzt eine Bildausschnitts-Voreinstellung, um Bilder ohne Skalierung zu verarbeiten.
  • Die Unterstützung für Patientendatensätze, die mit coDiagnostiX Version 9.0 oder älter geplant wurden, wurde verbessert.
  • Instabilitäten wurden in caseXchange, dem virtuellen Planungsexport, der Abutmentplanung und im PDF-Export behoben.

Italian

Comunicazione migliorata

  • Synergy ora aggiorna il modello clinico in coDiagnostiX se viene modificato in DWOS / CARES Visual.
  • Nella fase di design della guida chirurgica, il numero di ordine di caseXchange è ora disponibile come etichetta preimpostata. Il caso  deve essere aperto dalla Gestione ordini per accedere a questa impostazione.
  • Le e-mail di notifica caseXchange ora contengono ulteriori informazioni sul mittente.

Sistema di guaine/boccole

  • Aggiunto il supporto per il sistema di chirurgia guidata  Hahn™ Tapered Implant.
  • Aggiunto il supporto per il sistema di chirurgia guidata Medentika.
  • Supporto per gli impianti Straumann PURE aggiunto al sistema di chirurgia guidata Straumann.

Ulteriori miglioramenti

  • Quando si modificano gli abutment, la finestra di dialogo che chiede a quale impianto deve essere aggiunto l'abutment ora appare solo se rilevante.
  • La fase di importazione DICOM ora offre una preimpostazione di ritaglio dell'immagine per elaborare le immagini senza ridimensionamento.
  • Il supporto per i set di dati paziente pianificati con coDiagnostiX versione 9.0 o precedente è stato migliorato.
  • Le instabilità in caseXchange, nell'esportazione di pianificazione virtuale, nella pianificazione dell'abutment e nell'esportazione in PDF sono state corrette.
