Release Notes 9.6.5

English

  • Added support in caseXchange and order management for an additional identification number per contact (#4706)
  • Surgical protocol information for Straumann implants is provided to BienAir’s iChiropro (#4529)
  • Addresses some issues with the BienAir iChiropro link
  • Addresses an issue in the treatment evaluation module (#4727)
  • User information if Synergy connection partner uses another protocol version added (#4432)
  • Several other minor bugs fixed

Deutsch

  • caseXchange und die Auftragsverwaltung unterstützen nun eine zusätzliche Identifikationsnummer für Kontakte (#4706)
  • Für Straumann-Implantate wird das chirurgische Protokoll an BienAir iChiropro zusätzlich übertragen (#4529)
  • Weitere Korrekturen für die iChiropro-Verbindung hinzugefügt
  • Ein Fehler im Modul Behandlungsauswertung wurde korrigiert (#4727)
  • Hinweis hinzugefügt, wenn versucht wird, die Synergy-Verbindung mit einer älteren Version aufzubauen (#4432)
  • Weitere kleinere Fehlerbehebungen
