# 10. Support and service

coDiagnostiX Web is a demonstration re-implementation and has no distributor network. For
questions, defects and feature requests:

- **In-application help** — press **F1** anywhere; the help panel covers the current screen
  and links to this manual (`/manual`).
- **The manual sources** — `docs/manual/` in the repository; corrections are welcome as
  merge requests.
- **Issue tracker** — file reproducible reports (screenshots, the audit-log excerpt, browser
  and version from the About dialog) with the repository's issue tracker.
- **Version identification** — the About dialog (start screen → ⓘ) shows the application
  version and the third-party component list; quote it in every report.

For the trademark owner's original product and its certified support channels, see
chapter 1.2 — this project is not affiliated with them.
